GET 	/api/v1/1/msg 	
Получить последние сообщения.
Коды возврата:
	200 - Оk
	400 - Bad request
Выходной формат:
	{ 
	  "msgs" : [
			{ 
			  "msg_id" : 1, 
			  "room_id" : 3,
			  "msg_content" : "bla bla bla" 
			},
			{ 
			  "msg_id" : 2,
			  "room_id" : 5,
			  "msg_content" : "HELLO =)"
			} 
		   ]
	}


POST 	/api/v1/msg/new	
Отправить новое сообщение. Сообщение ограниченной длины, любого символьного содержания.
Коды возврата:
	200 - Оk
	400 - Bad request (отправка большей длины сообщения)
